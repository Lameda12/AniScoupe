## 📖 About

**AniScoupe** is a modern, beautifully designed web application built for anime enthusiasts. Discover new shows, keep track of your watch progress, and explore a world of anime directly from your browser. With a sleek UI powered by **Next.js 16**, **React 19**, and **Tailwind CSS**, AniScoupe delivers an immersive, lightning-fast experience.

## ✨ Features

- 🔍 **Discover & Search**: Find your next favorite anime using our real-time search and rich discovery layouts.
- 📋 **Personal Watchlist**: Add anime to your watchlist, track your progress, and manage what to watch next.
- 💬 **Quotes & Trivia**: Enjoy iconic quotes from your favorite shows within the app.
- 🎨 **Beautiful UI**: Highly polished, animated interface using Shadcn UI, Framer Motion, and Base UI.
- 🌙 **Dark Mode Optimized**: Easy on the eyes for those late-night binge sessions.

## 🚀 Getting Started

### Prerequisites
Make sure you have Node.js and npm (or pnpm/yarn/bun) installed.

### Installation

1. **Clone the repository:**
```bash
git clone https://github.com/Lameda12/AniScoupe.git
cd AniScoupe
```

2. **Install dependencies:**
```bash
npm install
# or yarn install / pnpm install / bun install
```

3. **Start the development server:**
```bash
npm run dev
# or yarn dev / pnpm dev / bun dev
```

4. **Open your browser:**
Navigate to [http://localhost:3000](http://localhost:3000) to see the app in action!

## 🛠️ Built With

* **Framework:** [Next.js (App Router)](https://nextjs.org/)
* **Library:** [React](https://reactjs.org/)
* **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
* **Animations:** [Framer Motion](https://www.framer.com/motion/)
* **UI Components:** [Shadcn UI](https://ui.shadcn.com/) & [Base UI](https://base-ui.com/)
